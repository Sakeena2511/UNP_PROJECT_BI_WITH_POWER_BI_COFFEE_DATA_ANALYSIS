# UNP_PROJECT_BI_WITH_POWER_BI_COFFEE_DATA_ANALYSIS

I Completed a Power-BI project in collaboration with UNP.This project focuses on analyzing coffee sales data using Microsoft Power BI to generate meaningful business insights. The objective of the project is to understand sales performance, customer purchasing behavior, product demand, and overall profitability using interactive dashboards and visual reports.Before creating visualizations, the data was:Cleaned (removed duplicates, handled missing values),Formatted (corrected date, currency, and numerical formats),Transformed using Power Query, Relationships were created between tables.Calculated measures were created such as:Total Sales = SUM(Sales Amount),Total Quantity Sold = SUM(Quantity),Average Sales per Order.The dashboard also includes important KPIs such as: Total Revenue, Total Orders, Total Quantity Sold, Average Order Value
